    [부정사용자 신고란?]
  • 구인구직 목적 이 외의 영업목적(자재납품, 보험, CCTV 등)의 연락을 받는 경우 신고하기를 통해 신고해주시면, 테라피닷컴에서 정보를 확인하여 테라피닷컴 회원이라면 영구이용정지로 이용 차단합니다.
    [신고방법]
  1. 마이페이지 > 부정사용자 신고 접속 후 좌측 하단에 신고하기 버튼을 클릭합니다.
  2. 부정사용자명, 부정사용자 연락처, 통화내역 스크린샷 첨부, 제목, 상세사유를 입력 후 하단 신고하기 버튼을 클릭합니다.
  3. 신고한 내용은 해당 메뉴에서 확인가능, 처리 완료 시 신고한 내용에 대해 답변을 드립니다.
